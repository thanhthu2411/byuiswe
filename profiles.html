Fitbit

The Fitbit project focused on building a wearable health monitoring system using embedded systems programming. Students integrated Bluetooth or wireless communication modules to collect and transmit real-time data. The project required scripting in languages like C or Python, working with APIs for data analysis, and documenting progress collaboratively. By the end, participants had hands-on experience with both hardware and software aspects of IoT devices.

Skills

Embedded Systems Programming
Bluetooth or Wireless Communication
Data Collection & Analysis
Scripting and Compiling Languages
API Integration
Project Documentation & Collaboration

SWE Website

This project involved designing and developing the official website for the SWE society using HTML, CSS, and JavaScript. Students created responsive layouts, styled content for readability, and added interactive features using DOM manipulation. The project helped members enhance their frontend web development skills while working in teams to build a user-friendly platform that showcases events, projects, and team profiles.p>

3D Plant Pot

The 3D Plant Pot project combined technology and sustainability. Students programmed Arduino microcontrollers to read data from sensors monitoring soil moisture and light levels. They designed custom pot enclosures using 3D modeling software and printed them using campus resources. This project encouraged creative thinking in product design, embedded development, and environmentally conscious tech solutions.

Skills

Arduino programming
Sensor integration
3D modeling & printing
Product design & prototyping
Embedded system development
Sustainable tech integration
